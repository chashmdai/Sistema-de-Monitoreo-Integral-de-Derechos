# Identidad visual SMID

SMID significa **Sistema de Monitoreo Integral de Derechos**. La propuesta visual busca funcionar como una submarca compatible con la Defensoria de la Ninez, sin copiar su icono oficial.

## Concepto

- El anillo de color representa integralidad, diversidad de dimensiones de derecho y relacion con la paleta institucional.
- El centro usa una red/radar simple para comunicar monitoreo, trazabilidad y coordinacion.
- La palabra **SMID** se trata como nombre principal; el descriptor completo aclara el significado para personas no tecnicas.

## Archivos

- `smid-logo-concept-v1.png`: propuesta generada como referencia visual original.
- `smid-logo-transparent-light.png`: version PNG con fondo transparente para fondos claros.
- `smid-logo-transparent-dark.png`: version PNG con fondo transparente para fondos oscuros.

El dashboard usa copias optimizadas en `scripts/dashboard/assets/`:

- `smid-logo-light.png`
- `smid-logo-dark.png`

## Uso recomendado

Usar la version clara sobre fondos blancos o muy claros, y la version oscura sobre fondos oscuros. Mantener SMID como sistema asociado, no como reemplazo del logo oficial de la Defensoria.
