Para empezar

  1. Levanta MySQL (y RabbitMQ si pruebas el flujo de eventos).
  2. Pulsa Diagnóstico para verificar entorno y coherencia del JWT.
  3. Pulsa Levantar todo: arranca por niveles (infra → núcleo → gateway).
  4. Usa Obtener token para autenticarte y probar las rutas /api/*.

«Levantar todo» inicia el ecosistema operativo: base, SMID Core, ESNNA, SGS, Gateway y frontend. El check agrega servicios opcionales que siguen en laboratorio.
